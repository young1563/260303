용어 사전 설계 배경 및 목표

게임 산업은 데이터, 기술, 예술이 융합된 복합적인 분야입니다. 이 페이지는 실무 현장에서 사용되는 핵심 언어들을 정리하여 빠르고 정확한 커뮤니케이션을 돕기 위해 설계되었습니다.

🎯

커뮤니케이션 최적화

공통된 용어 정의를 통해 오해를 줄이고 업무 효율성을 극대화합니다.

📈

지표 중심의 기획

단순한 재미를 넘어 데이터에 근거한 의사결정 역량을 기릅니다.

🚀

실무 프로세스 이해

기획부터 라이브 운영까지 전체 제작 파이프라인의 언어를 익힙니다.

🔍

비즈니스 및 분석 지표

11

DAU

User

Daily Active User. 일간 활성 사용자 수. 하루 동안 게임에 접속한 유저의 총합입니다.

중요도: ★★★

NRU

User

Newly Registered User. 신규 가입 사용자 수.

중요도: ★★★

Retention

User

잔존율. 특정 일자에 앱을 설치한 유저 중 다시 방문한 유저의 비율입니다. (D-1 리텐션 등)

중요도: ★★★

IAP

Revenue

In-App Purchase. 인앱 결제. 유저가 앱 내에서 유료 결제를 통해 추가 콘텐츠를 구매하는 행위입니다.

중요도: ★★

ARPDAU

Revenue

Average Revenue Per Daily Active User. 활성 유저 1명당 평균 결제 금액입니다.

중요도: ★★★

ARPPU

Revenue

Average Revenue Per Paying User. 유료 결제 유저 1명당 평균 결제 금액입니다. 가격 밸런스를 볼 때 중요합니다.

중요도: ★★

LTV

Revenue

Lifetime Value. 유저가 게임 시작부터 이탈까지 지출하는 총 기대 금액을 뜻합니다.

중요도: ★★★

App Events

Data

앱 내에서 발생하는 특정 행위(클리어, 구매 등)를 서버로 전송하여 통계에 반영하는 이벤트입니다.

중요도: ★★★

A/B Test

Data

유저를 두 그룹으로 나누어 기존 버전과 테스트하고자 하는 변경안을 비교하여 효율을 확인하는 기법입니다.

중요도: ★★★

ROAS

Marketing

Return on Ad Spending. 광고비 대비 매출 수익률입니다.

중요도: ★★★

Viral

User

유저의 자발적인 포스팅이나 친구 초대 등을 통한 자연스러운 전파 행위입니다.

중요도: ★★★

마케팅 및 운영

13

Organic User

Marketing

자연 유입 유저. 광고를 통하지 않고 스스로 검색이나 차트를 통해 들어온 유저를 뜻합니다.

중요도: ★★★

Ad Network

Marketing

광고주와 매체를 연결해 주는 플랫폼 회사입니다. (AdMob, AppLovin 등)

중요도: ★★

Cross Promotion

Marketing

이미 운영 중인 자사 게임끼리 유저를 교차 홍보하여 공유하는 마케팅 방식입니다.

중요도: ★★

Mediation

Marketing

여러 광고 매체의 효율을 비교 분석하여 가장 수익이 높은 쪽의 광고를 우선 노출해 주는 시스템입니다.

중요도: ★★★

CPI

Marketing

Cost per Install. 앱 설치 1건당 광고주가 지불하는 단가입니다.

중요도: ★★★

CPA

Marketing

Cost per Action. 특정 액션(IAP 등)을 취한 유저에 대해 지불하는 단가입니다.

중요도: ★★

eCPM

Marketing

Effective Cost per Mille. 유료 광고 노출 1,000회당 광고 수익 효율을 의미합니다.

중요도: ★★★

CTR

Marketing

Click Through Ratio. 광고 노출 대비 클릭되는 비율입니다.

중요도: ★★★

ASO

Marketing

App Store Optimization. 앱 스토어 내 검색 결과에서 노출 순위를 높이기 위한 최적화 작업입니다.

중요도: ★★★

Localization

Marketing

현지화. 특정 국가의 언어, 문화, 환경에 맞게 제품을 맞추는 과정입니다.

중요도: ★★

Re-Engagement

Marketing

접속하지 않는 휴면 유저를 다시 게임으로 불러오기 위해 진행하는 마케팅 활동입니다.

중요도: ★

Influencer Marketing

Marketing

소셜 플랫폼의 영향력 있는 크리에이터를 활용하여 진행하는 광고입니다.

중요도: ★

CS

Operation

Customer Service. 고객 관리 서비스. 불만 처리나 스토어 평점 관리 등의 업무를 포함합니다.

중요도: ★★★

게임 기획 및 메커닉

13

Match-3

Genre

같은 블록을 3개 이상 매칭 시켜 제거하는 장르의 게임입니다.

중요도: ★★★

Collapse

Genre

인접한 같은 색상의 블록을 클릭하여 제거하는 장르의 게임입니다.

중요도: ★★★

Line-matching

Genre

블록을 드래그하여 연결해 매칭 및 제거하는 방식의 게임입니다.

중요도: ★★★

Stage

Structure

레벨들의 묶음 단위입니다. 보통 한 스테이지에 10~15개의 레벨이 포함됩니다.

중요도: ★★★

Lobby

System

게임 진입 전의 대기 화면. 보통 월드맵, 상점, 옵션 UI 등이 배치됩니다.

중요도: ★★★

In-Game

System

실제 유저가 게임 플레이를 즐기는 물리적 게임 영역을 뜻합니다.

중요도: ★★★

Out-Game

System

인게임 외의 모든 시스템(퀘스트, 상점, 공지 등)을 아우르는 명칭입니다.

중요도: ★★

Level Update

Ops

이미 서비스 중인 게임에 주기적으로 신규 레벨들을 추가하는 업데이트 행위입니다.

중요도: ★★★

Build Update

Ops

기능 수정이나 에셋 변경 등을 포함한 새 버전을 스토어에 배포하는 행위입니다.

중요도: ★★★

Push Notification

System

스마트폰 상에서 유저의 접속을 유도하기 위해 보내는 알림 기능(로컬/서버 푸시)입니다.

중요도: ★★★

Invite

System

유저가 자신의 친구에게 우리 게임으로의 초대를 보내는 행위입니다.

중요도: ★★★

Casual Game

Genre

간단한 조작으로 짧은 시간 동안 쉽게 즐길 수 있는 가벼운 게임 타입입니다.

중요도: ★★

Store Rating

Feedback

스토어에 남겨진 유저들의 평점과 리뷰입니다. 앱 스토어 최적화에 중요한 영향을 미칩니다.

중요도: ★★

그래픽 및 UI/UX

14

Modeling

Graphic

3D 툴을 사용하여 오브젝트를 제작하는 과정입니다.

중요도: ★★★

Vector / Bitmap

Graphic

점과 선으로 이루어진 벡터와 픽셀 단위의 비트맵 이미지 차이입니다.

중요도: ★★★

Spine

Animation

2D 본 애니메이션 툴로, 자연스러운 메쉬 움직임을 구현할 때 사용합니다.

중요도: ★★★

Particle System

Effect

불꽃, 연기 등 수많은 작은 입자를 제어하여 특수 효과를 만드는 시스템입니다.

중요도: ★★★

Optimization

Tech

최적화. 리소스 용량 최소화 등을 통해 원활한 실행 환경을 만드는 작업입니다.

중요도: ★★★

UI / UX

Design

조작 화면 구성(UI)과 더불어 유저가 느끼는 경험 전반(UX)을 의미합니다.

중요도: ★★★

UGUI

Unity

유니티에서 UI 리소스를 제작하기 위해 보편적으로 사용하는 엔진 내 GUI 툴입니다.

중요도: ★★★

9 Slice

Design

이미지 리소스를 9조각으로 나누어 모서리는 유지하고 중앙만 늘려 최적화하는 기법입니다.

중요도: ★★

Dummy Image

Dev

제작 완료 전 가체결 상태로 배치하는 임시 테스트용 이미지입니다.

중요도: ★★

Layout

Design

제한된 공간 안에 각 그래픽 요소를 효율적으로 배치하고 디자인하는 작업입니다.

중요도: ★★

Creative

Marketing

광고 모델링이나 배너, 영상 등 마케팅 전략에 사용되는 모든 시각 에셋을 뜻합니다.

중요도: ★★★

Blur Effect

Tech

이미지를 뿌옇게 처리하여 시각적인 집중도나 심도를 조절하는 효과입니다.

중요도: ★

Typography

Design

폰트의 서체, 배치, 구조를 통해 텍스트의 정보를 시각적으로 전달하는 행위입니다.

중요도: ★

Loader Image

UI

앱 로딩이나 데이터 통신 시 유저에게 첫 노출되는 메인 로더 이미지입니다.

중요도: ★

사운드 및 영상

7

BGM

Sound

Back Ground Music. 게임의 무드와 감정을 결정하는 배경음악입니다.

중요도: ★★★

SFX

Sound

Sound Effect. 버튼 클릭이나 폭발 등 특정 행동에 반응하는 효과음입니다.

중요도: ★★★

Jingle

Sound

짧은 멜로디성 효과음으로, 레벨 클리어나 패배 시 사용됩니다.

중요도: ★★★

Loop

Sound

음악의 처음과 끝 지점을 연결하여 끊김 없이 반복 재생하는 기법입니다.

중요도: ★★

Normalize

Sound

들쭉날쭉한 사운드 리소스의 음량을 일정하게 맞추는 공정입니다.

중요도: ★★

Store Preview

Video

앱 스토어 페이지에서 게임 플레이를 미리 볼 수 있도록 구성한 영상입니다.

중요도: ★★★

UA Video

Video

User Acquisition Video. 신규 유저 모객을 위해 제작되는 광고 홍보 영상입니다.

중요도: ★★★
